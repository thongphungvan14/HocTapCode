Bài tập buổi 1: Viết 1 bài giải phương trình bậc 2 (cho cả vd nha)

Cách giải phương trình bậc 2:

Bước 1: Tính Δ = b 2 - 4ac

Bước 2: So sánh Δ với 0

  • Δ < 0 phương trình (1) vô nghiệm
  • Δ = => 0 phương trình (1) có nghiệm kép x1 = x2 = -b⁄2a
  • Δ > 0 phương trình (1) có 2 nghiệm phân biệt, ta dùng công thức nghiệm sau:

    x1 = (-b + √Δ)⁄2a và x2 = (-b - √Δ)⁄2a

    • Ví dụ:

    Giải phương trình 4x2 - 2x - 6 = 0

    Δ = (-2)2 - 4.4.(-6) = 4 + 96 = 100 > 0 => phương trình (2) đã cho có 2 nghiệm phân biệt.

    x1 = (-(-2)2 + √100)⁄2.4 = 3⁄2

    x2 = (-(-2)2 - √100)⁄2.4 = -1
